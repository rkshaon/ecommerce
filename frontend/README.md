# E-Commerce Frontend

## Project requirements

Node version
```
v18.12.1
```
npm version
```
9.6.7
```

## Project setup

### Step 1:

Go to the frontend directory.

```
cd frontend
```

### Step 2:

Install dependencies.
```
npm install
```

### Step 3:

Update value `API_BASE_URL` [@config.js](./src/config.js) based where you have run backend.

### Step 4:

Run frontend server.
```
npm run serve
```
